#[zapwut, `!?`, %zpwt](#zpwt)

[Short description]

#Syntax

    `!?`, `zapwut`, `[%zpwt p=twig]` is a synthetic hoon that
enforces a Hoon version restriction.

##Produces

[Twig or tile]

##Sample

[`p` is a _
`q` is a _]

##Tall form

To declare code that runs only in Hoon 164K or newer:

    !?  164
    q

To declare code that runs only in 164K through 161K:

    !?  [164 161] 
    q

##Wide form

None

None
