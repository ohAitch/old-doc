#buccab `$_` %bccb

Default value ([Bunt]())

`$_` is a synthetic rune used to produce the default value ([bunt]]()) of a given tile. `$_` is one of the four [tile reductions]() in hoon.

##Produces

[Twig]()

##Sample

`p` is a [tile]().

##Tall form

    $_  p

##Wide form

    $_(p)

##Irregular form

    _p
