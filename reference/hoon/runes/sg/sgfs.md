#Syntax

`~/`, `sigfas`, `[%sgfs p=term q=twig]` is a synthetic hoon that
implements one common case - a gate arm in a book, ie, a library
function - of the `%sgcn` jet hint.  `%sgfs` assumes the parent
axis is `7` and there are no children.
