#[cenzaz, %cnzz](#cnzz)

Pull wing `p`

`cenzaz` is a synthetic hoon that pulls wing `p` from the subject. `cenzaz`  is only used internally by `hoon.hoon`, but is used implicitly whenever assignment occurs.

##Produces

Twig: `[%cnzz p=wing]`

##Sample

`p` is a [`++wing`]()
